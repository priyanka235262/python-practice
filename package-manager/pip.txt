pip:

Purpose: A package installer for Python. It's used to download, install, and manage third-party Python packages from repositories.
Functionality:
Installs packages from PyPI by default.
Can install packages from local files or URLs.
Can uninstall packages.
Can list installed packages.
Can update installed packages.
Supports virtual environments for isolating project dependencies.
Usage:
Basic installation: pip install <package_name>
Uninstallation: pip uninstall <package_name>
Listing installed packages: pip list
Updating installed packages: pip install --upgrade <package_name>
Key Features:
Easy to use and intuitive command-line interface.
Supports various package formats (e.g., wheels, source distributions).
Can be customized with configuration files.
Integrates well with virtual environments.
